TaskHub

TaskHub is a modern task-management application built with React, Redux Toolkit, Material UI, i18next, and React Hook Form.  
It allows users to create columns and tasks, move tasks between workflow stages, switch between light/dark mode, and toggle the language between English and Persian.

Features
1- Create columns and tasks  
2- Move tasks between different workflow stages (drag/drop style workflow)  
3- Light / Dark mode toggle  
4- Bilingual support (English / Persian) using i18next  
5- Responsive UI with Material UI  
6- Login and Sign Up pages with full form validation

Technologies and Libraries
1- React  
2- Redux Toolkit  
3- Material UI (MUI)  
4- React Hook Form  
5- i18next

How to Run:
1- npm install  
2- npm run dev
